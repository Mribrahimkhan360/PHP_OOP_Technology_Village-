# 📘 OOP Concept Learning

## 📌 Description

This repository is created to learn and understand the core concepts of Object-Oriented Programming (OOP).  
It contains simple examples and explanations of fundamental OOP principles using practical code examples.

The goal of this project is to build a strong foundation in OOP for software development.

---

## 🎯 Objectives

- Understand the basics of OOP
- Learn how to create classes and objects
- Implement real-world OOP examples
- Improve coding structure and reusability

---

## 🧠 OOP Concepts Covered

### 1️⃣ Class
A blueprint for creating objects.

### 2️⃣ Object
An instance of a class.

### 3️⃣ Encapsulation
Wrapping data and methods together inside a class.

### 4️⃣ Abstraction
Hiding complex implementation details and showing only essential features.

### 5️⃣ Inheritance
A mechanism where one class inherits properties and methods from another class.

### 6️⃣ Polymorphism
The ability of a method to behave differently based on the object.
